# 沟通 Communication

良好的沟通是实现长期稳定财富管理道路不可或缺的部分。由于个人财富管理目标与财务支出规划匹配，而财务支出规划与家庭生活密不可分。虽然与长辈、伴侣、子女沟通财务支出规划并达成一致是一件非常具有挑战性的任务，但这并不是放弃沟通的借口。

- 让每次沟通交流尽可能简短易懂，不要因为具体事项的争执而迁移到其他方面以免引起更多不必要的争执。
- 尽可能按照工作流程，厘清财富管理和财务支出的细节，提前制定意外事件的应对方案，详细记录并备份。后续执行时严格按照计划，不要轻易因为一时情绪而偏离计划。
- 当无法立即达成一致时，暂时搁置争议，转移话题或转换沟通环境，等双方情绪降温后或选择其他合适的时间继续沟通。
