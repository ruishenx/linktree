# 典型的断章取义（持续更新）

> 标注部分通常都被刻意忽略导致误解。

不听老人言，吃亏在眼前。<mark>尽信老人言，终身未向前。</mark>

撑死胆大的，饿死胆小的。<mark>淹死会水的，打死犟嘴的。</mark>

<mark>父慈</mark>子孝，<mark>兄良</mark>弟悌，<mark>夫义</mark>妇听，<mark>长惠</mark>幼顺，<mark>君仁</mark>臣忠。

富贵险中求，<mark>也在险中丢</mark>。

父母在，不远游，<mark>游必有方</mark>。

可怜之人必有可恨之处，<mark>可恨之人必有可悲之处</mark>。

人情留一线，日后好相见。<mark>斩草不除根，春风吹又生。</mark>

忍一时风平浪静，退一步海阔天空。<mark>忍无可忍，无需再忍。</mark>

听人劝，吃饱饭。<mark>人云亦云，难成大器。</mark>

未经他人苦，莫劝他人善。<mark>你若经我苦，未必有我善。</mark>
