# 无风险共担的行为具有脆弱性

几种典型的没有做到风险共担的情形：

- 他人给出誓言、观点、建议等却没有任何抵押物（信用在这种情况下不算抵押物，因为对方失信时很难对其进行有效制裁，财产是最有效的抵押物），此时建议是脆弱的。
- 教师教授的理论与学者撰写的论文，相关人员自己都不相信这些知识内容，此时这些知识内容是脆弱的。
- 媒体发表虚假信息带来的曝光度大于监管对其的惩罚，此时出自媒体的信息是脆弱的。
- 金融机构依靠投资者资金的抽成盈利而不依靠所投资产的增值，此时其所有交易行为的可靠性都是脆弱的。
- 立法部门制定的法律、司法部门裁决的案件、执法部门开展的行动，相关人员自己都不相信是在按照公开规则执行，此时这些部门的所有对外声明都是脆弱的。
- 政治集团与企业集团的领导在短任期内制定长周期政策与方案，此时政策与方案的合理性是脆弱的。
- 政治集团宣扬廉洁透明却不公开全部政治过程与财产分布，此时廉洁透明度是脆弱的。
- 政治集团中的权贵家族依靠逃税积累财富，此时强制缴税的道德性与法理性是脆弱的。
- 发动战争的统治者，其家人亲属不直接在前线参战，此时统治者的战争决策合理性是脆弱的。
- 中国大陆、俄罗斯、朝鲜、伊朗等地区的政治集团在本土宣传美国、欧盟、英国、加拿大、澳大利亚、日本、韩国、台湾等地区“危险”，却将家人亲属安置在这些地区，此时宣传内容是脆弱的。
- ...

> 无风险共担 → 激励扭曲 → 信息失真 → 责任外溢 → 信任坍塌 → 系统脆弱

<mark>【注意】无风险共担是脆弱的充分条件，即无风险共担一定具有脆弱性，但脆弱性不一定都无风险共担。</mark>

强制风险共担的措施：

- **抵押机制**：决策者必须承担对称风险。
- **延迟收益机制**：收益分期支付与失败回溯扣回。
- **共损结构**：无限连带责任与终身信誉绑定。
- **信息透明与可验证**：高透明度并不是风险共担，但可以降低风险外包转移空间。
