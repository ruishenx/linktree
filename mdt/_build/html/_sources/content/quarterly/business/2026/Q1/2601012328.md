# 系统性转变与无希望的未来

## 五种系统性转变

随着数据、算力、算法的发展壮大，由计算系统引发的至少 5 种可预见的系统性转变即将到来。

### 物理通用操作系统

工业的发展方向将从拥有工程经验的人类转移到由机器系统控制的数字孪生。物理通用操作系统的目的是让现实物理世界可被软件持续调度、更新、回滚。

AI 控制系统 + 仿真模拟系统 + 传感器，将数字孪生从“展示工具”变成“控制接口”，所有制造、基建、物流等行业将全面数字化。

虽然可以预见发展方向，但是目前依然存在巨大的短板就是拥有现实物理世界约束条件的高质量数据依然稀缺，强化学习算法生成的数据大量不可用。

### 合成生物工业化

生物与化学的发展方向将让实验室的试错控制权从人类转移到机器系统。生化产品的闭环流程是“假设 → 实验 → 失败 → 更新”，机器系统逐渐接管流程后，可用产品的制造效率将会大幅提高，曾经极其漫长的周期将会大幅缩短。所有材料、医药、食品等行业将会产生质的飞跃。

虽然可以遇见发展方向，但是目前依然存在巨大的自动化不确定性，整个流程难以收敛，因为研发流程的约束条件对于人类来说依然存在众多未解之谜。

### 轨道空间商业化

SpaceX 以一己之力让物质从地球表面搬运至地球轨道空间的价格大幅下降，轨道空间的基建、平台、应用将会在可预见的未来快速展开。

### 抽象意识数字化

LLM 的大规模商业应用尤其是在语言领域已经通过了 Turing test，这并不代表 LLM 生成文字的方式与人类相同，而是展现出一种形式，即在模拟人类的语言和情感等抽象内容是可以通过数据、算力、算法来衡量的， LLM 通过 token 来量化这些内容。

这种人类高级抽象能力可被量化的研究目前依然属于伦理与道德灰色区域甚至禁区，但是技术总会向着更深的方向探索，利用机器对人类高级抽象能力的数字化模拟将会导致人类意识也成为商品变得可定价和可交易。

### 博弈与决策系统化

复杂度日渐上升的博弈与决策已经超出人类生理承受的极限，因此引入机器系统作为辅助成为常态。这是一种人类将责任下放给机器系统的现象，这不是一种选择而是妥协。

Palantir 的平台属于目前全球领先的此类产品，它将多元数据对齐融合，生成决策链路，让博弈和决策从抽象内容成为可被机器系统执行的内容。

在可预见的未来，此类系统在军事、政治治理、金融等行业的普及将会带来革命性的变化。

## 无希望的未来

站在个人立场，可以由自己控制的未来被称为“有希望”。但是随着以上 5 种甚至更多由于全体人类为了更大的欲望而将权力下放给机器系统的现象称为常态，由数据、算力、算法驱动的机器就将控制个人的未来，这将是“无希望”。

再也没有任何神秘和科幻，一切都将由数学公式控制，拥有机器系统控制权的相当少比例的个人将成为人类群体的绝对控制者。对于被控制者，他们的未来没有希望，因为被机器系统控制；对于控制者，他们为了保证个人不沦落到被控制者的行列，需要自控。整个人类群体都均衡在一种个人世界的自洽闭环中。

无论是反抗还是拥抱，通向未来的时间线都不会改变。
