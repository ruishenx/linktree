# 两性 Gender

## 女性 Woman

- 牡丹 Peony

端庄大气，拥有该气质的女性，能帮助伴侣在事业道路上稳步前行，但因可以独当一面就要求伴侣始终不断进取。

![](./img/peony.jpg)

- 桃花 Peach Blossom

甜美活力，拥有该气质的女性，能带给伴侣青春的动力，但因岁月的流逝可能会导致气质逐渐消散。

![](./img/peach_blossom.jpg)

- 红玫瑰 Red Rose

明艳娇媚，拥有该气质的女性，能带给伴侣显著的感官享受，但也容易招致外界的干扰。

![](./img/red_rose.jpg)

- 白樱花 White Sakura

清冷温婉，拥有该气质的女性，能带给伴侣显著的内心享受，但也容易导致精力内耗。

![](./img/white_sakura.jpg)

## 男性 Man

- 海盗 Pirate

自由，拥有该特质的男性，能带给伴侣广阔的精神享受，但容易遭遇不稳定的冲击。

![](./img/pirate.png)

- 教授 Professor

智慧，拥有该特质的男性，能带给伴侣显著的精神依靠，但容易被禁锢在封闭的思想状态中。

![](./img/professor.png)

- 狼人 Werewolf

强壮，拥有该气质的男性，能带给伴侣显著的感官依靠，但容易目光狭隘性格焦躁。

![](./img/werewolf.png)

- 吸血鬼 Count Dracula

高贵，拥有该特质的男性，能帮助伴侣实现阶级地位的提升，但容易营造巨大的压力。

![](./img/count_dracula.png)
